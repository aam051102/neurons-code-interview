# Development
## Back-end
- `cd back-end`
- `python manage.py runserver`

## Front-end
- `cd front-end`
- `npm run dev`

This opens the site on localhost:3000.

# Tests
## Back-end
- `cd back-end`
- `python manage.py test`

## Front-end
- `cd front-end`
- `npm run cypress:open` for full interface
    - OR `npm run cypress:component:headless` for quick run of component tests
